01 Основные определения термодинамики

В тетради: "Введение" - 03.09.2020

Термодинамическая система - совокупность материальных тел, находящихся во взаимодействии между собой.

Термодинамическая система:

  • Открытая. Обменивается с окружающей средой.
  • Закрытая. Обмен не происходит.

Теплоизолированная термодинамическая система - система, в к.т. не происходит обмен тепловой энергией с окружающей средой.

Механически изолированная термодинамическая система - система, в к.т. не происходит обмен механической энергией с окружающей средой.

Все тела, не входящие в систему, называются внешними или окружающей средой.

Для изучения тепловых машин применяют промежуточные тела, называемые рабочими телами.

В теории рабочее тело - это идеальный газ.

Идеальный газ - любой газ, при применение которого можно пренебречь силами взаимодействия между молекулами, а молекулы представляют в виде материальных точек.

02 Параметры идеальных газов

В тетради: "Основные параметры рабочего тела" - 03.09.2020

  1. Давление (P). Давление в сосуде - средний результат количества ударов молекул о стенки сосуда.
    Единицы измерения: 1 Па = 1Н/м2 [СИ];
    кПа = 103 Па; мПа6 = 10 Па.

    Давление:

    • Атмосферное (барометрическое). 1 бар = 105 Па, давление атмосферного воздуха у поверхности Земли, измеряемое барометром.
    • Манометрическое (избыточное). В сосуде, если P газа > атмосферного давления.
    • Вакуумное(разреженное). Если P газа < атмосферного.

    Ни атмосферное, ни манометрическое давления не являются параметрами рабочего тела. За параметры рабочего тела принимаются абсолютное давление, к.т. определяется следующим образом:

    - если Pизб. > Pатм., то Pабс. = Pизб. + Pатм.;

    - если Pразр. < Pатм., то Pабс. = Pатм. - Pразр..

    1 мм. рт. ст. = 133 Па; 1 мм. вод. ст. = 9,8 Па.

  2. Температура

    (t, T). Температура определяет степень нагретости тела. Измеряется косвенным путём, например, по степени расширения жидкости:
    t℃, T(К), T = t + 273К.
  3. Удельный объём - объём единицы массы вещества: ν = v/m (м3/кг;
  4. Плотность вещества (ρ) - масса в единицах объёма: ρ = m/v (кг/м3)

03 Законы идеальных газов

В тетради: "Законы идеальных газов" - 04.09.2020
I закон.
Закон Бойля-Мариотта.

Опытным путём установлено, что при переходе газа из одного состояния в другое при постоянной t соотношение между давлением и удельным объёмом запишется следующим образом:

Для 1 кг газа: P1/V2 = ν2/ν1; T = Const;
Для M кг: P1/V2 = V2/V1.
II закон.
Закон Гей-Люссака. Для 1 кг газа: T1/T2 = ν 1/ν 2; P = Const (изобарный);
Для M кг T1/T2 = V1/V2.
III закон.
Закон Шарля

Опытным путём установлено, что при переходе газа из данного состояния другое, соотношение между давением и t будет записано следующий образом:

T1/T2 = P1/P2; ν V = const (изохорный);
IV закон
Объединённый газовый закон.

При условии, что меняются все параметры рабочего тела, соотноешние между параметрами запишется следующим образом:

Для 1 кг: P1ν/T1 = P2ν2/T2;
Для M кг: P1V1/T1 = P2V2/T2.

V закон.
Закон Авогадро

Закон Авогадро может быть оформлен следующим образом: в равных объёмах разных газов при одинаковом давлении и температуре содержится одинаковое количеств молекул.

P1 / P2 = M1/M2

Плотность двух газов пропорциональна их молярной массе, т.к. плотность является обратной величиной удельного объёма. M(моль)

P = 1/ν => ν2/ν1 = M1/M2, M1ν1 = M2ν2 = M3ν2=Mnνn = const.

Масса газа, выраженная в кг, численно равная молярной массе, называется киломольной. km = Mν - называется 1кМ и измеряется в [м3/кмоль].

Для разных газов при нормальных условиях: P = 760 мм.рт.ст; T = 237К, объём кмоля одинаков и сотсавляет 22,4(м3/М); ρн = M/22,4
Уравнения состояния идеальных газов

Для любого состояния идеальных газов соотношение между параметрами рабочего тела определяется однозначно.

Для 1 кг: Pν = RT
Для M кг: MRT
Для 1 кмоля: PVM = мRT - универсальная газовая постоянная; mR = 8314 [Дж/кмоль*К];
R = 8314/M - газовая постоянная.

Для вычисления численных значений во все законы идеальных газов и уравнения состояний подставляются параметры рабочего тела.

04 Газовые смеси

В тетради "Газовые смеси" - 07.09.2020
I. Способ задания газовой смеси по массе.

Газовой смесью называют совокупность газов, находящиххся в одном объёме и невступающих в реакцию между собой. Любой входящий в состав смеси газ называется компонентом.

Если рассматривать отдельно компонент смеси, то он создаёт на стенке сосуда определённое давление. Это давление называют парциальным. Давленние смеси определяется: Pсм = P1 + P2 + P3 + ... + Pn = Σn1;
где P1;P2;P3;PN - парциальные давленимя компонентов смеси.

Представтим, что смесь состоит из 3х компонентов, для которых известны массы. Тогда масса смеси = M1 + M2 + M3.

Отношение массы компонента к массе смеси называется долей компонента смеси: M1/Mm = g1; M2/Mm = g2; M3/Mm = g3.

Массовую долю можно выражать в %. 100% = q1+q2+q3+gn=Σn1 gn; 1 = q1+q2+q3+gn=Σn1 gn.

II. Способ задания газовой смеси по объёму.

Объём, который занимает компонент смеси, при условии, что давление компонента = давлению смеси и t компонента = t смеси, называется парциальным объёмом(Pк = Pм)

Объём смеси определяется суммой всех объёмов помпонентов смеси: Vсм = V1+V2+V3+VN =Σn1Vn;

Отношение объёма коммонента кк объъёму смеси называется объёмной долей: V1/Vм = r1, V2/Vm = r2, V3/Vм = r, 100% = r1+r2+r3; rn=Σn1rn; 1 = r1+r2+r3+rn=Σn1rn

10 Теплоемкость идеальных газов

Теплоемкость газов это количество теплоты, которое необходимо сообщить единице количества газа, чтобы нагреть её на 1 градус. Обозначается буквой C.
Теплоемкость может быть истинной и средней.

  • Истиной теплоемкостью называется отношение количества теплоты, сообщенной газом в элементарном процессе при бесконечном малом изменении температуры.
    C=dq/dt
  • Средняя теплоемкость - количество теплоты, которое в среднем расходуется, для нагрева 1 кг. газа на 1 градус.
    Сm = 1/t1-t2

11 Эквивалентность теплоты и работы; I-й закон термодинамики

В тетради: Первый закон термодинамики. Эквивалентность теплоты и работы. - 10.09.2020

На основании различных опытов учёными в XIX в. был открыт закон сохранения и превращения энергии:

Энергия не исчезает бесследно и не возникает из ниоткуда, а превращается из одного вида в другой в
строго эквивалентных количествах.

Этот закон в области взаимного превращения в тепловой механической энергии был назван первым закон термодинамики.

Если к телу подвести некоторое количество теплоты, то оно совершит эквивалентную работу. Таким образом можно это записать:

Q = α [кДж]; α = Q[кДж], где

Q - количество подведённой теплоты, α - совершённая работа.

12 Определения термодинамического процесса

Любые изменения параметров рабочего тела называют термодинамическим процессом(нагрев, охлаждение, расширение, сжатие).

Расширение: P уменьшается, V увеличивается;
Сжатие: P возрастает, V уменьшается.

Если в процессе параметры успевают выровняться по всему объёму газа в каждый момент времени, то такое состояние называют равновесным.

Процесс, состоящий из бескончно большого количества равновесных состояний называется равновесным процессом.

Равновесные процессы можно изобразить в виде графика с помощью двух параметров: P и V.

Если в процессах расширения и сжатия рабочее тело проходит через одни и те же промежуточные состояния(расширение 1a-2, сжатие 2a-1), то такие процессы называются обратимыми.

Если же в процессе рабочее тело идёт по линии 1a-2, а в обратном процессе по линии 2b-a, то такой процесс называется необратимым.

Все реальные процессы считаются в физике необратимыми, т.е. обратимые процессы возможны только для идеальных газов.

13 Внутренняя энергия идеальных газов

В тетради: "3. Внутренняя энергия идеального газа" - 11.09.2020

Внутренняя энергия газа складывается из его кинетической и потенциальной энергии, зависит от t и P(для реальных газов).

Т.к. для идеальног газа отсутствует сила взаимодействия между молекулами, то нутренняя энергия складывается только из потенциальной энергии рабочего тела и зависит только от t.

Т.к. температура является параметром рабочего тела, то и внутренняя энергия тоже является его параметром.

Внутренняя энергия обозначается буквой U [кДж/кг].

Изменение внутренней энергии: ΔU = U1 - U2

14 Работа идеальных газов

В тетради: "4. Работа идеального газа" - 11.09.2020

Работа газами совершается только при изменении объёма.

dl = (P*F*dS)/ dv;
dl = Pdv, где F - площадь поршня, dS - бесконечно малый путь, на к.т.сдвинулся поршень, P - давление(всегда положительное), dv - изменение объёма, dl - работа поршня.

Если объём поршня > 0, то совершённая поршнем работа будет >0.

dv > 0 => dL > 0; - расширение;
dv > 0 => dL < 0; - сжатие.

Для определения работы в произвольном процессе, разделим на бесконечно большое количество элементарных процессов, для которых dV – элементарное изменение V (бесконечно малое) для каждого элементарного процесса А определиться по формуле: dl= PdV и будет эквивалентно площади под элементарным процессом, а для всего процесса А определиться суммой этих элементарных работ.

L = ∫v1v2pdv – формула определения всего процесса.

15 Математическое выражение I-го закона термодинамики

Математическое выражение первого закона термодинамики в дифференциальной форме иногда представляют так:

dU = σQ - pdV - σA'

Последнее выражение включает A′ – «полезную» работу, т.е. все виды работы, не связанные с изменением объема. Например, «полезная» работа за счет электрических сил, химических превращений и т.д.

Для открытых термодинамических систем математическое выражение первого закона имеет вид:

ΔU = Q - A + Em;
dU = σQ - σA + dEm.
где Em - энергия, связаннаяс изменением массы термодинамической системы.

Рассмотрим любой процесс при подводе теплоты.

Нагревая рабочие можно сказать, что если изменилось температура рабочего тела (или газа), то изменилась внутренняя энергия, а если изменился V, то совершилась работа, ∆U = U1-U2 если же не каких изменений не происходит, то можно записать вся теплота тратится на изменение работы и внутренней энергии.

Q = α + ∆U следовательно dq= dα+dU, где q – количества теплоты.
α – работа

78 Гидродинамический и тепловой пограничный слой

Рассмотрим, как происходит движение жидкости вдоль твёрдой поверхности.

При движении жидкости вдоль пластины, те частицы которые непосредственно соприкасаются с ней на некотором расстоянии от начала пластины, останавливаются (как бы прилипают к ней): т.е ωж = 0

Любая реальная жидкость обладает вязкостью, поэтому 2-й слой контактирующий с прилипшим слоем, движется вдоль него и тормозится им. На 2-й слой набегает 3-ий, он проходит чуть дальше 2-го и тоже тормозится им. На 3-й слой набегает 4-й, затем 5-й и так далее. До тех пор, пока не появится слой, скорость которого будет равна средней скорости жидкости.

Слой жидкости у поверхности теплообмена, скорость которого изменяется от 0 до средней скорости жидкости называется гидродинамическим пограничным слоем.

Рассмотрим, как изменится t жидкости у поверхности теплообмена.

Слой прилипший к поверхности имеет t равную t ст, он не движется и теплота передаётся теплопроводностью.

Второй слой получает некоторое кол-во теплоты, часть её уносится потоком, а остальное передаётся след. слою, т.е в 3-ий слой теплоты поступают меньше, в 4-й ещё меньше и т.д.

До тех пор, пока не появится слой, в которой теплоты не поступилло и его Lж = Lср.

Слой жидкости у поверхности теплообмена в котором t меняется, от Lсm до tср называется тепловой пограничный слой.

87 Теплоотдача при кипении

Процесс перехода жидкости в пар называется кипением.

Для поддерживания процесса кипения необходимы следующие условия: tст>tн.

α = q/(tст - tн)

Рассмотрим как происходит кипение жидкости в сосуде большого обьёма и как изменяется α.

p>Для осуществления процесса конденсации необходимо соблюдать условия tсм < tн.

В результате конденсации выделяется некоторое количество теплоты, которое необходимо отвести от жидкости.

Конденсация зависит от чистоты поверхности, расположении и формы поверхности.

АВ – теплоотдача при свободном движении жидкости;
БВ – теплоотдача при пузырьковом кипении;
ГД – теплоотдача при плёночном режиме кипения;
ВГ – 1ый кризис при кипении;
СЖ – 2ой кризис при кипении.

В начальный момент времени, когда температурный напор(tн – tж) достаточно большой, теплоотдача осуществляется за счет свободного движения жидкости участка АБ. С увеличением теплового потока у поверхности теплообмена начинают образовываться пузырьки пара, которые постепенно возрастают и отрываясь, поднимаются вверх. Возникает пузырьковое кипние жидкости.

Пузырьковое кипение это кипение жидкости с образованием пузырьков пара.

При пузырьковом кипении коэф. теплоотдачи достаточно большой, в какой-то момент у поверхности теплообмена может образовываться сплошная паровая плёнка, которая определяет основную жидкость от поверхности теплообмена.

α резко падает (участок ВГ) -1ый кризис при кипении. Если поверхность не разрушиться, то продолжится режим плёночного кипения. (участок ГД).

Кипение жидкости с образование сплошной паровой плёнки у поверхности теплообмена называется пленочным режимом кипения.

При уменьшении теплового потока может произойти переход от плёночного режима к пузырьковому (участок СЖ) - 2ой кризис при кипении

В этом случае α меньше, чем при кризисе кипения.

92 Теплообмен излучения между телами положенными одно в другом

Рассмотрим, как происходит теплообмен излучением между телами, расположенными одно в другом

Для тел известно: T1 и T2; поверхность теплообмена F1 и F2, коэф. излучения C1 и C2, поглощающая способность тел A1 и A2.

В процессе теплообмена вся энергия с первого тела попадает на поверхность второго. С поверхности второго только часть энергии β попадает на поверхность первого тела, остальная энергия(1-β) попадает на собственную поверхность 2-го тела.

Определим количество лучистой энергии, которая излучается первым и вторым телом.

Q1 = C1 * (T1/100)4 * F1 + (1-A) * Q * β
        -------.-------     ----------.----------
Q2 = C2 * (T2/100)4 * F2 + (1-A) * Q1 + (1-β) * (1-A2) * Q
        -------.-------               ---------------.---------------
     собственное              отражённая энергия
    излучение тел
Если T1 > T2, то Q = Q1 - Q2.
